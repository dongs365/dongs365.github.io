docker命令 本地镜像管理

docker 本地镜像查看、删除、修改、build、历史、导出、导入。

docker images

列出本地镜像。

语法

OPTIONS说明：

docker rmi

删除本地一个或多少镜像。

语法

OPTIONS说明：

docker tag

标记本地镜像，将其归入某一仓库。

语法

docker build

命令用于使用 Dockerfile 创建镜像。

语法

OPTIONS说明：

docker history

查看指定镜像的创建历史。

语法

OPTIONS说明：

docker save

将指定镜像保存成 tar 归档文件。

语法

OPTIONS说明：

docker import

从归档文件中创建镜像。
