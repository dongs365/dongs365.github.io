docker命令 镜像仓库

linux
  • docker

docker 仓库登陆、登出、镜像拉取、推送、搜索。

docker login

登陆到一个Docker镜像仓库，如果未指定镜像仓库地址，默认为官方仓库 Docker Hub

docker logout

登出一个Docker镜像仓库，如果未指定镜像仓库地址，默认为官方仓库 Docker Hub

语法

OPTIONS说明：

docker pull

从镜像仓库中拉取或者更新指定镜像

语法

OPTIONS说明：

docker push

将本地的镜像上传到镜像仓库,要先登陆到镜像仓库

语法

OPTIONS说明：

docker search

从Docker Hub查找镜像
